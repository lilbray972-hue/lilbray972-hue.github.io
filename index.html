YtbPOST is a Windows desktop tool for managing and scheduling bulk YouTube video uploads across multiple channels. Built for content creators who need efficiency at scale.

View on GitHub Learn More

Bulk Upload

Upload multiple videos simultaneously with customizable metadata — titles, descriptions, tags, and thumbnails — all from a single interface.

Scheduled Publishing

Schedule uploads in advance with precise timing. Queue videos across channels and let YtbPOST handle the rest automatically.

Multi-Channel Management

Manage uploads across multiple YouTube channels from one dashboard. Switch between accounts seamlessly without re-authentication.

Desktop-Native

Built with Python and PySide6 for Windows. No browser required, no cloud dependency. Your credentials and data stay on your machine.

GPU-Accelerated

Leverages NVIDIA NVENC for hardware-accelerated video processing. Faster encoding and transcoding before upload.

YouTube API Compliant

Uses the official YouTube Data API v3 for all operations. OAuth 2.0 authentication, proper quota management, and full Terms of Service compliance.
